      \begin{column}{0.495\linewidth}
        \begin{block}{BACKGROUND}
          Clinicians managing complex patients must manually navigate multiple
          longitudinal test reports. It is a cognitively demanding process
          that makes it difficult to identify trends and synthesise findings
          at the point of care. The burden of integrating these findings can
          lead to missed insights and suboptimal clinical decisions.
        \end{block}
      \end{column}

      \begin{column}{0.010\linewidth}\end{column}

      \begin{column}{0.495\linewidth}
        \begin{block}{AIMS}
          To develop a system that automatically extracts, summarises, and
          visualises trends from longitudinal patient test reports (structured
          and unstructured data), and surfaces personalised best-practice
          clinical recommendations.
        \end{block}
      \end{column}

    \begin{block}{METHODS}
      The system uses a model-agnostic LLM to iteratively process patients'
      test reports, extract data points for key clinical metrics, plot trends
      and create a summary. The prompt specified instructions to parse
      reports, expected metrics and guardrails to avoid hallucinations.
      Best-practice recommendations were codified and dynamically presented
      to the user through a rule-based look-up mechanism, personalised for
      the patient. The system was tested on respiratory function tests and
      echocardiograms.

    \end{block}

    \vfill

    \begin{block}{RESULTS}
      The system analysed reports, extracted key information, presented
      trends over time in plot and table formats, provided text-based
      summaries and gave best-practice clinical recommendations. Results
      were reviewed by an experienced anaesthetist and found to be accurate.
    \end{block}

    \vfill

    \begin{block}{CONCLUSIONS}
      This proof-of-concept demonstrates that a GenAI-based system can
      accurately synthesise longitudinal patient reports, visualise results
      and surface personalised best-practice recommendations, significantly
      reducing clinician cognitive load. A prospective clinical trial is
      warranted.
    \end{block}

      \end{itemize}
    \end{block}
